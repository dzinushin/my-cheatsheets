


## Mac OS terminal commands:


### scutil

```
scutil --get ComputerName
sudo scutil --set ComputerName banana
scutil --dns
```

### iponfig

```
ipconfig ifcount
ipconfig getifaddr en0
```

### arch
how change arch for command (install on M1 binary with arch x86_64)
```
arch -x86_64 zsh 
nvm install 14
```
